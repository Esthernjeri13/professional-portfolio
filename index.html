Lab Challenges

REST API Optimization Challenge
Problem Statement:

Improve the response time of a Django REST API handling high volumes of ticket booking requests.

Approach:
  • Analyzed existing API endpoints with Django Debug Toolbar
  • Implemented database query optimization
  • Added caching with Redis
  • Experimented with FastAPI for comparison
Tools Used:
Python Django FastAPI Redis
Key Lessons:
  • N+1 query problem significantly impacts performance
  • Caching can reduce response times by up to 70% for read-heavy endpoints
  • FastAPI shows promise for high-performance use cases
CTF: Secure Hostel Management System
Problem Statement:

Identify and fix security vulnerabilities in the Hostel Management System web application.

Approach:
  • Performed penetration testing using OWASP ZAP
  • Identified SQL injection and XSS vulnerabilities
  • Implemented input validation and parameterized queries
  • Added CSRF protection and secure session management
Tools Used:
OWASP ZAP Node.js Express MySQL
Key Lessons:
  • Never trust user input - always validate and sanitize
  • Security headers are essential for modern web apps
  • Regular security audits should be part of the development lifecycle
